O que é o Construct 3?

É um software de desenvolvimento de jogos que roda diretamente no navegador, sendo usado por escolas, desenvolvedores independentes e empresas em todo o mundo. O Construct 3 oferece uma forma fácil, rápida e visual de criar jogos 2D que podem ser exportados para plataformas móveis, desktops e web.
